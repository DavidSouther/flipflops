# Flip Flops

## Finish Blogging. Go to the beach.

Take a little syntax out of your life. It's just a file system and your words.

## Quickstart

```bash
npm install --global flipflops
mkdir myproject
cd myproject
flipflops init
flipflops serve
```

Visit [http://localhost:1024/](http://localhost:1024/)
